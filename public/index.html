Easy to install!

Want to install MagicMirror² on your freshly installed Raspbery Pi? Just run the following bash command:

curl -sL https://raw.githubusercontent.com/MichMich/MagicMirror/master/installers/raspberry.sh | bash

Prefer a manual installation? Check out the installation guide!

Interested in developing your own modules?

MagicMirror² has an extensive documentated API. It allows you to built your own module backed by a powerfull backend.
Check out the API documentation for more information and start developing today.
